🍔 FoodHub — Food Delivery Management System

A full-stack web application for managing food delivery operations, built with Java Servlets, JSP, MySQL, and Maven. The system supports multiple user roles — customers, restaurants, delivery agents, and admins — each with their own dedicated interface and functionality.



✨ Features

Customer registration, login, profile management, and order tracking
Restaurant dashboard with menu management (add, edit, delete items)
Order placement with cart, order history, and real-time status updates
Delivery Agent registration, login, and order assignment dashboard
Payment processing and payment history
Reviews system for completed orders
Admin dashboard with analytics, customer list, and platform-wide stats
Role-based access control (Customer / Restaurant / Driver / Admin)
Session management for secure page access
